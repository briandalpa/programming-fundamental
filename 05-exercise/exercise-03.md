# Exercise Instructions

## 1. Return Primitive Data Types Only

Write a function based on the array below that will return **primitive data types only**.

**Example:**

```js
let arr = [1, [], undefined, {}, "string", {}, []];
// Output → [1, undefined, "string"]
```

## 2. Sum of Duplicate Values

Write a function from the array below that will return the **sum of duplicate values**.

**Example:**

```js
let arr = [10, 20, 40, 10, 50, 30, 10, 60, 10];
// Output → 40
```
