# Exercise Instructions

## 1. Sum of Numbers in a Mixed Array

Write a function from a given array of mixed data types and return the sum of all the numbers.

**Example:**

```js
const mixedArray = ["3", 1, "string", null, false, undefined, 2];
// Output → 3
```

## 2. Insert Integers with Maximum Size Limit

Write a function to insert multiple given integers (not an array) into an array, with a defined maximum size. If the maximum size of the given input is 5, the array can only contain 5 elements.

**Example:**

```js
const maxSize = 5; // given integers is 5, 10, 24, 3, 6, 7, 8
// Output → [5, 10, 24, 3, 6]
```

## 3. Combine Two Arrays

Write a function that will combine two given arrays into one array.

**Example:**

```js
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
// Output → [1, 2, 3, 4, 5, 6]
```

## 4. Find Duplicates in an Array

Write a function to find duplicate values in an array.

**Example:**

```js
const arr = [1, 2, 2, 2, 3, 3, 4, 5, 5];
// Output → [2, 3, 5]
```

## 5. Find Difference Between Two Arrays

Write a function to find the difference between two given arrays.

**Example:**

```js
const arr1 = [1, 2, 3, 4, 5];
const arr2 = [3, 4, 5, 6, 7];
// Output → [1, 2, 6, 7]
```
